# dotfiles

Dotfiles for arch linux (STILL TESTING)

## Installation

```bash
 git clone --recurse-submodules https://github.com/xDDoubleTea/dotfiles ~/dotfiles
 cd dotfiles/
 chmod +x ./setup.sh
 ./setup.sh
```

Reboot if necessary.

You're all setup! (STILL TESTING)
